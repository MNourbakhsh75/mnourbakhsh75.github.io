Notable Projects

Emotion Recognition of Speech

• Emotion recognition from sounds using a variety of machine learning techniques was undertaken. A dataset containing sounds depicting various emotions, such as Anger, Disgust, and Sadness, was assembled. From the data, diverse features like Chroma, Constant Q-transform, and Zero Crossing Rate were extracted.
• These features were subjected to different classification techniques such as KNN, SVM, and MLP for emotion classification. Furthermore, various clustering methods, including K-Means, DBSCAN, and K-Medoids, were applied to the data to gain a more comprehensive understanding of its underlying structure.

BERT Implementation From Scratch  

• The implementation of Bidirectional Encoder Representations from Transformers (BERT) was carried out. To attain a deeper comprehension of their core functionality, all components of a Transformer, including the encoder, decoder, and multihead attention mechanisms, were constructed from the ground up. Additionally, Segment Embeddings, Token Embeddings, and Position Embeddings were developed as part of the implementation process. Moreover, an Attention Visualizer was created using the bertviz library.

GAN Implementation  

• This assignment involved the implementation of various Generative Adversarial Networks (GANs). Specifically, a Deep Convolutional GAN was built from the ground up, and various techniques were employed to enhance its training process. These techniques included One-sided label smoothing, batch normalization, and the addition of noise. Furthermore, a Wasserstein GAN (WGAN) was implemented, incorporating different loss functions aimed at enhancing the overall performance of the GAN.

Working with Testnet Network

• The project’s objective was to acquire insight into the mechanics of Bitcoin. It involved generating a valid address for the testnet network utilizing SHA-256 and RIPEMD-160 hash functions. Additionally, through the utilization of python-bitcoinlib, various transactions were created and disseminated within the testnet network. Furthermore, the project encompassed the simulation and testing of the complete mining process using an actual block from the Bitcoin mainnet network.

Job offer System
